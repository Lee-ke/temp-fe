+   基本步骤

    +   进入目录：`cd xx`
    +   创建目录：`mkdir xx`
    +   从远程下载 git 项目：`git clone ...`
    +   提交到远程:
        
        ```bash
        git add .
        git commit -m "备注"
        git push origin master
        ```
